## 📖 عن المشروع (About The Project)
**Neto (نيتو)** هو نظام متكامل لإدارة الشؤون المالية (PFM) مصمم خصيصاً لتلبية احتياجات المستقلين (Freelancers)، والعاملين بنظام الدخل المتعدد. يعالج النظام مشكلة غياب التخطيط المالي لدى أكثر من 60% من المستقلين، ويعتمد على تقنيات الذكاء الاصطناعي لتقديم تحليلات دقيقة، تتبع الدخل والمصروفات، وتقديم نصائح مالية ذكية تساعدك على اتخاذ قرارات مبنية على بيانات واضحة وتنمية أعمالك بثقة.

## ✨ المميزات الرئيسية (Key Features)
* 💰 **تتبع مالي دقيق:** تسجيل وتتبع الدخل والمصروفات من مصادر متعددة وتصنيفها تلقائياً.
* 🧠 **تحليلات مدعومة بالذكاء الاصطناعي:** دمج واجهات الذكاء الاصطناعي (Groq / Gemini API) لتقديم نصائح مالية مخصصة وقراءات تنبؤية.
* 👥 **إدارة العملاء والفواتير:** إصدار فواتير احترافية مخصصة (Custom Branded Invoicing) ومتابعة حسابات العملاء.
* 📊 **تقارير دورية وتخطيط ضريبي:** أدوات مخصصة لتقدير الضرائب وإصدار تقارير شهرية وسنوية.
* 🎨 **تجربة مستخدم سلسة (UI/UX):** واجهة عصرية، تفاعلية، وسهلة الاستخدام تناسب المحترفين.

## 🎯 الجمهور المستهدف (Target Audience)
* الفريلانسرز وصناع المحتوى (يوتيوب، تيك توك، إنستجرام).
* العاملين بنظام الدخل المزدوج (مرتب ثابت + فريلانس).
* أصحاب المشاريع الصغيرة (SMEs).

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **Backend:** C# / .NET
* **Frontend:** React /tailwind css
* **AI Integration:** Gemini API / Groq API
* **Architecture:** 3-Tier Architecture / RESTful APIs
* **Database:** SQL Server
